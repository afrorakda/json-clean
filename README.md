# Json-clean

Part of the Fridge Combo project. A minimalist JSON formatter to make minified data readable instantly.

## 1 Tool / 1 Action

Clean up messy JSON code. **Json-clean** takes a single string of minified JSON and transforms it into a perfectly indented, human-readable structure with one click or paste.

## Philosophy
* **1 Tool, 1 Action.**
* **No Login / No Ads / No Tracking**
* **Browser-only Privacy:** Your data never leaves your browser. Safe for handling sensitive configuration strings.

## How to Use
1. **Paste:** Paste your messy or minified JSON into the workspace.
2. **Auto-Format:** The tool instantly expands and indents the code for readability.
3. **Error Check:** If the JSON is invalid, the border turns red and displays "Invalid JSON".
4. **Copy:** Tap [ Copy Clean JSON ] to grab the formatted result.

## Features
* **Smart Validation:** Instantly identifies syntax errors.
* **Compact Design:** Optimized for low-spec devices and quick tasks.
